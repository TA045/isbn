# 📕 本の表紙ダウンローダー (Book Cover Downloader)

ISBN（13桁）を入力するだけで、本の表紙画像を取得し、ダウンロードできるシンプルなWebアプリケーションです。
書誌情報（タイトル・著者名）を自動で取得し、分かりやすいファイル名で保存できます。

## 主な機能

- **ISBNによる書籍検索**: 13桁のISBNを入力すると、リアルタイムで書籍情報を検索します。
- **表紙画像の表示とダウンロード**: 見つかった本の表紙を大きく表示し、クリックひとつでダウンロードできます。
- **書誌情報の自動取得**: [openbd](https://openbd.jp/)と[Google Books API](https://developers.google.com/books)を利用して、本のタイトルと著者名を自動で取得します。
- **ファイル名の自動生成**: ダウンロードするファイル名は`<ISBN>_<タイトル>_<著者>.jpg`のように、管理しやすい形式で自動生成されます。
- **検索履歴**: 一度検索した本の表紙は履歴として一覧表示され、履歴から直接ダウンロードしたり、不要な履歴を削除したりできます。
- **依存関係なし**: このツールは単一のHTMLファイルで完結しており、特別なライブラリやフレームワークを必要としません。

## 使い方

1.  このリポジトリにある`index.html`（またはお手元のHTMLファイル）を、お使いのWebブラウザで開きます。
2.  `ISBN を入力`と書かれた入力欄に、調べたい本の13桁のISBN（`978`から始まる数字）を入力します。
3.  入力を進めると、自動的に書籍情報と表紙画像の検索が開始されます。
4.  表示されたメインの表紙画像、または履歴欄の画像をクリックすると、ファイルがダウンロードされます。

## 使用技術・API

- HTML
- CSS
- JavaScript (ES6+ / async/await)

### 外部API

- **書誌情報取得**:
  - openbd API
  - Google Books API
- **表紙画像取得**:
  - Amazon.co.jp Product Images

## 注意事項

- このツールはWebサーバーを必要とせず、ローカルのHTMLファイルを直接ブラウザで開くことで動作します。
- APIから情報を取得するため、インターネット接続が必要です。
- すべての書籍で情報や画像が取得できるとは限りません。APIに登録されていない、または画像が存在しない場合があります。
- Amazonの画像取得URLの仕様は将来的に変更される可能性があり、その場合は画像が表示されなくなることがあります。
